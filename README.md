# Knowledge Base

This repository is a shared document store organized by function. Each directory contains documentation relevant to that team or domain.

## Structure

| Directory | Purpose |
|-----------|---------|
| [product/](product/) | Product strategy, roadmap, personas, and release process |
| [engineer/](engineer/) | Technical architecture, onboarding, and deployment |
| [gtm/](gtm/) | Go-to-market positioning, messaging, and launch planning |
| [miscellaneous/](miscellaneous/) | Cross-functional references, meeting notes, and vendor records |

## Contributing

Add new documents to the most relevant directory and update that directory's `README.md` table with a one-line description.
